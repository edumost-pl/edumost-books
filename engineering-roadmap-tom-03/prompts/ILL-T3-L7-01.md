Лестница DNS-запроса

**Тип иллюстрации:**  
Образовательная схема · вертикальная цепочка · кэш сбоку

**Главная цель иллюстрации:**  
Показать **полный путь DNS**: ноутбук → Pi-hole (радужное кольцо) → облако upstream → здание «авторитетный DNS»; **сбоку кэш** — полка с коробками (иконки глобуса + **полоска TTL** — без цифр).

Что подросток должен почувствовать: **«каждый уровень что-то знает»** — рекурсия понятна визуально.

---

**Описание сцены**

**Вертикальная «лестница»** по центру кадра (снизу вверх):

1. **Ноутбук** героя (внизу) — иконка **вопросительного облачка** над ним (запрос — **без слов**)
2. **Pi-hole** — радужное кольцо на Pi
3. **Облако upstream** — серое с **тремя точками** (1.1.1.1 — **без цифр**, только форма облака)
4. **Здание-сервер** «авторитетный DNS» — стилизованный дата-блок

**Сбоку справа** — **полка-кэш**: 3 коробки с иконкой глобуса и **цветной полоской** (TTL) — **без имён доменов**.

Герой сидит внизу слева, **указывает** на лестницу (или смотрит на схему на планшете **без текста**).

Стрелки между уровнями — **янтарные** (↑ вверх, ↓ ответ).

**Что НЕ должно появляться:** `google.com`, `nas.home`, `dig` output, цифры IP.
